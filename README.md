# JavaScript Programming Project

A web application developed while completing the JavaScript Programming course at Western Governors University. This project demonstrates modern front-end development techniques using JavaScript and Angular to create a responsive, interactive user experience.

## Overview

This application showcases front-end development concepts including component-based architecture, dynamic data binding, and user interaction. It was designed to strengthen skills in building modern web applications using industry-standard tools and frameworks.

## Features

- Interactive user interface
- Responsive design
- Dynamic content rendering
- Component-based architecture
- Client-side routing
- Data binding and event handling

## Technologies Used

- Angular
- TypeScript
- JavaScript
- HTML5
- CSS3
